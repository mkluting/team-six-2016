# Voyage Manager

A Node.js web application intended to run on Raspberry Pi hardware.

# Requirements

 - [Node.js][1] `>= 4.2.6`

# Getting Started

```bash
# Optional, deps are checked into the repository
$ npm install

# Run server.
# Navigate to http://localhost:8080 in your favorite browser!
$ node app.js
```

[1]: https://nodejs.org/en/
